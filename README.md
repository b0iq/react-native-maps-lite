# react-native-maps-lite

Performant and minimal map view for react-native

## Installation

```sh
npm install react-native-maps-lite
```

## Usage

```js
import { MapsLiteView } from "react-native-maps-lite";

// ...

<MapsLiteView color="tomato" />
```
